# -- Laboratorio Bloque 4 - Simulador Jurassic Park --
## Java + Python de Axity 2026

Este proyecto es el motor central de un simulador de parque temático desarrollado en **Java 17** y potenciado con **Spring Boot** y **Lombok**.
Simula la interacción de los turistas con las zonas del parque, así como la de los trabajadores con
las mismas zonas y los dinosaurios. Aplica fuertemente principios de POO y todo lo aprendido durante el curso.

El proyecto aún está incompleto, hasta el punto de escribir este README en esta versión, apenas se ha
finalizado la fase 1 de modelado.

Sin embargo, ya incluye dependencias para la persistencia de datos con H2 y Liquibase.
